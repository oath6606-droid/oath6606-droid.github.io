# 卡通贪吃蛇小游戏（静态网页）

纯前端实现的贪吃蛇小游戏，卡通风格，支持键盘与触屏操作，可直接用浏览器打开本地文件游玩。

## 运行方式

1. 确认本目录下包含：
   - `index.html`
   - `style.css`
   - `script.js`
2. 双击 `index.html`，使用现代浏览器（Chrome / Edge / Firefox 等）打开即可，无需任何部署或安装。

> 如果浏览器有本地脚本安全限制，建议使用一个本地静态服务器（如 VS Code 的 Live Server 插件）来打开。

## 操作说明

- **电脑键盘**
  - 方向：`↑ ↓ ← →` 或 `W A S D`
  - 开始 / 暂停：`空格`
- **手机 / 平板**
  - 点击右侧方向按钮控制
  - 在游戏区域内**滑动**控制方向，**轻点**可以暂停 / 开始

## 游戏规则

- 吃到一个水果：长度 +1，得分 +10。
- 每吃到 5 个水果，速度会提升一档（最多 8 档）。
- 撞到墙壁或自己的身体则游戏结束。
- 页面自动记录本地最高分（使用 `localStorage`）。
